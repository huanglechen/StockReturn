# StockReturn

Project files for the ECE C147A stock return prediction work.

## Recommended Entry Points

- `PROJECT_SUMMARY.md`: current project overview and latest averaged results
- `C147A_project_average.ipynb`: main multi-seed notebook with outputs
- `C147A_project_seeded.ipynb`: fixed-seed notebook with outputs
- `C147A_project_nonseeded.ipynb`: single-run development notebook
- `run_project_mhattn.py`: main script for attention-based experiments
- `run_project_attention_ts.py`: related time-series attention experiment
- `stock_return_core.py`: shared core library
- `C147_proposal.pdf`: original project proposal

## Notes
- Historical scripts and earlier prototypes are stored under `archive/`.
